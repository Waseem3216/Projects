# Resume Bullets for XYMZ Docker Project

Use one of these depending on how much detail you want on your resume.

## Technical resume bullet

- Dockerized a retro search-engine web portal using NGINX, Node.js, and Docker Compose; configured reverse proxy routing, container health checks, and an isolated bridge network for frontend-to-API communication.

## Short resume bullet

- Containerized a frontend and API web application with Docker Compose, NGINX reverse proxying, and service health checks.

## Project description

**XYMZ Dockerized Search Portal** is a portfolio web application that demonstrates containerized deployment of a static frontend and lightweight backend API. The frontend is served through NGINX, while Docker Compose orchestrates the frontend and API services on a private bridge network.

- Docker Compose orchestration

- NGINX reverse proxy configuration
